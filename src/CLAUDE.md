# src/ — Клиентские модули

Каждая папка — отдельный клиент с собственным CLAUDE.md, промптами, скриптами и данными.

- `config_manager.py` — общий менеджер конфигурации, читает `clients_config.json`
- `{client}/CLAUDE.md` — контекст и правила для конкретного клиента
- `{client}/docs/PROMPT.md` — главный промпт-инструкция для Claude по этому клиенту
